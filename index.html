F 基本面 (0-5)

  • 成長 (0-2)：營收 YoY、EPS YoY 趨勢
  • 品質 (0-2)：FCF 體質、獲利穩定性
  • 估值 (0-1)：Forward P/E 預期差
觀察指標：P/E · P/S · PEG · FCF Margin · D/E · ROE

T 技術面 (0-5)

  • 趨勢 (0-2)：50D/200D 均線方向、多空判讀
  • 風報比 (0-2)：關鍵支撐壓力位、52W 區間
  • 量價 (0-1)：突破放量或量縮整理確認
趨勢判讀：多頭(價>200D且50D上穿) · 空頭(價<200D且50D下彎) · 盤整(均線糾結)

R 風險 (0-5，越高越危險)

  • 波動回撤 (0-2)：Beta、最大回撤幅度
  • 不確定性 (0-2)：財報可見度、市場預期
  • 事件風險 (0-1)：監管/訴訟/地緣等
注意：風險分數越高代表越危險，與基本/技術面相反

行動決策規則

加倉條件
基本面 ≥ 4 且 技術面 ≥ 3 且 風險 ≤ 2
或明確支撐位分批小加
減倉條件
風險 ≥ 4
或 技術面 ≤ 1 且 基本面 ≤ 2
或 估值偏貴且近壓力位
平倉條件
基本面 ≤ 1
或 風險 ≥ 4 且 結構破位/敘事崩塌
每檔輸出： 結論(加/減/平+信心) → 持倉快照 → 估值/成長/FCF/負債表 → 評分(F/T/R) → 基本面重點 → 技術面觀點 → 催化劑 → 風控停損 → 待補資料
