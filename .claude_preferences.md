# Jerry 的 Claude 使用偏好

## 自動建立 Skill 規則
如果一個工作流程是**可重複操作**的（例如相同步驟可套用到不同輸入），Claude 應該**預設主動將它做成 Skill**，不需要使用者額外要求。判斷標準：
- 流程包含 3 個以上固定步驟
- 可套用到不同的輸入資料（不同 Ticker、不同檔案等）
- 輸出格式是固定的

## 語言偏好
- 分析內容（reason、key_theme）使用**繁體中文**
- JSON 欄位名稱、category slug 維持英文
- 對話語言以繁體中文為主

## 已建立的 Skills
- **news-analyzer**: 股票新聞情緒分析，輸入 `{TICKER}.json` → 輸出 `{TICKER}_analysis.json`
